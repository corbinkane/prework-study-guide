HTML

  • Hypertext Markup Language or HTML is used to create the structure of a webpage
  • the head element contains information about the webpage.
  • the body element represents the visible content shown to the user
  • HTML is made up of elements which are a type of content on a webpage that provide structure and framework
  • The head element contains information "(metadata)" this inofrmation is not able to be seen by the user and helps the page load correctly
  • The body element contains all of the content that is visible to the user
  • Declarations are not case-sensitive but HTML elements are case-sensitive
  • The declaration we use is < !DOCTYPE html > this tells the browser what type of document it will be using
  • Attributes define aditional information about an element and HTML attributes always go in the beginning element never the closing element
  • The title element defines the tile of the webpage it is essential and helps with Search Engine Optimization or SEO which is the process of proving the ranking of a website in search engines
  • The style element defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The meta element defines metadata or information such as the character set, description, keywords, author, and viewpoint. For other information stored in the meta tag visit CLICK HERE
  • The script element embeds client-side scripts in the HTML document
  • If you want to learn more about UTF-8(variable-length character encoding) and character encoding CLICK HERE
  • Within the body there are other elements this creates the parent child language with the body being the parent and the other elements being the children
  • The body element is an example of a semantic HTML element which is an element that describes it content and the role it plays on the webpage
  • Visible HTML elements is contained between an opening tag and a closing tag
  • The elements h1, h2, h3, h4, h5, and h6 are headings to help organize a page with h1 being the biggest heading and h6 being the smallest
  • ul, ol, and li are elements that represent unordered lists, ordered lists, and list items
  • the img elements contains information about images on the webpage and contains various attributes the src attribute being the most important because it defines the images location
  • the br element creats a line break between two blocks of content
  • To create a link on a website you use the a element which stands for anchor followed by href = and then the url

CSS

  • There are three types of CSS used to style a webpage inline CSS, internal CSS style sheet, and external CSS style sheet
  • Inline CSS is used to style a particular page element only and are less flexible than internal and external style sheets an example would look like < h1 style = "color : red;" > prework study guide < / h1 >
  • An internal style sheet contains the CSS rules for the webpage in the head element and only apply to that page an example would look like
    < head >
    < style >
    h1 { color : red ; }
    < / style >
    < / head >
  • An external style sheet is the most common way because it allows developer to keep all of their CSS rules in a seperate file which makes design changes easier but you do need to use a link placed in the head element like this example
    < head >
    < link rel = "stylesheet" href = "./assets/style.css" />
    < / head >
